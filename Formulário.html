Para que serve um formulário e qual sua função?

Um formulário pode ser ultilizado para varias funções,sendo sua marioria,formulários HTML são um dos principais pontos de interação entre um usuário e um web site ou aplicativo.
Eles permitem que os usuários enviem dados para o web site. Na maior parte do tempo, os dados são enviados para o servidor da web, mas a página da web também pode interceptar para usá-los por conta própria.
